# Opencode

- https://github.com/sst/opencode
- https://opencode.ai/docs

## ¿Qué puede hacer OpenCode?

- Leer archivos
- Editar archivos
- Ejecutar comandos
- Integración con Git
- Trabajar con APIs


---

# Installation

---

## Installation

```bash
curl -fsSL https://opencode.ai/install | bash
```

---

## Installation

```bash
curl -fsSL https://opencode.ai/install | bash
opencode auth login
```

---


### Quickstart

```bash
opencode
opencode models
opencode init
```

[!TIP] Use `/init` to initialize a new project or `/help` to see available
commands.

---

### Advanced


```bash
opencode agent create
ls ~/.config/opencode/agent
```

> [!NOTE]
> Read documentation
> https://opencode.ai/docs/agents/#markdown
